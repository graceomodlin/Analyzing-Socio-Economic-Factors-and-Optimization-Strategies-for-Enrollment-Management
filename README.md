# Analyzing-Socio-Economic-Factors-and-Optimization-Strategies-for-Enrollment-Management
This research project explores the integration of data science and operation research methodologies to predict university tuition rates and maximize institutional profit. 

U.S. university dataset from 2001 to 2022 from kaggle was used for this project. https://www.kaggle.com/datasets/ryusonoda/u-s-university-dataset-from-2001-to-2022 
This is a dataset found on Kaggle composed of data gathered by an organization called the College Scorecard from colleges and universities across the country.
